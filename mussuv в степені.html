Вам дано масив із додатними числами та числом N. Ви повинні знайти N-ту потужність елемента в масиві з індексом N. Якщо N знаходиться поза масивом, поверніть -1. Не забувайте, що перший елемент має індекс 0. Давайте розглянемо кілька прикладів: масив = [1, 2, 3, 4] та N = 2, тоді результат 3 ^ 2 == 9; масив = [1, 2, 3] та N = 3, але N знаходиться поза масивом, тож результат -1. https://www.codewars.com/kata/57d814e4950d8489720008db/train/javascript
